Ejercicio 1

Para permitir el muestreo ascendente, es necesario modificar la función para que asigne un cero a las entradas que no sean enteras.

figure;
f = @(n) exp(-n/5).*cos(pi*n/5).*(n>=0).*(n==fix(n));
n = -10:10; stem(n,f(n/2),'k.'); ylabel('f[n/2]'); xlabel('n'); axis([-10.5 10.5 -0.5 1.1]);

El término agregado (n==fix(n)) es uno si n es un número entero y cero si n no es un número entero. el modificado La función es fácil de probar.

figure;

Ejercicio 3

Para empezar, observamos que x(t) comienza en t = π. Dado que 2−t decae a menos de 1/100 de su resistencia original en 7 segundos, es suficiente representar x(t) en t = 10.

Dado que xo(t) incluye x(t) y su reflexión, a el rango de tiempo adecuado para nuestra gráfica es −10 ≤ t ≤ 10.

 u = @(t) 1.0*(t>=0); t = -10:1/20:10;

Dado que cos(2πt) oscila un ciclo por segundo, a una densidad de trama de 20 puntos por segundo es suficiente para una trama de calidad razonable.

 x = @(t) 2.^(-t).*cos(2*pi*t).*u(t-pi); xo = @(t) (x(t)-x(-t))/2;

Usamos la funcion plot con las variables previamente designadas y graficamos:

 plot(t,xo(t),'k-'); xlabel('t'); ylabel('x_o(t)'); grid on;
 axis([-10 10 -1/20 1/20]); set(gca,'xtick',[-10 -pi 0 pi 10]);

Ejercicio 4

Problema 3.1-1 c Find the energy of the signal depicted in Fig.P3.1-1c

Por la naturaleza de la señal podemos definir este intervalo

n = -3:3;

sympref('HeavisideAtOrigin',1);

Hacemos anónima a la funcion

xn = @(n) 3*n.*(heaviside(n+3)-heaviside(n-4));

Revisando el ejercicio 3 la formula para el cálculo general de la energía de una señal discreta finita será: E=(sum(f(n).^2) con el intervalo n definido Aplicamos la formula para la señal xn, y obtenemos

Exn = sum(xn(n).^2);
